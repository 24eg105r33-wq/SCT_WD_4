# To-Do App

## Project Overview

This project is a colorful and interactive To-Do application developed using HTML, CSS, and JavaScript. It helps users manage their daily tasks by adding, organizing, editing, and tracking completed tasks.

## Features

- Add new tasks
- Organize tasks into different lists/categories
- Set date and time for tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Filter tasks:
  - All tasks
  - Completed tasks
  - Pending tasks
- Data stored using Local Storage
- Responsive and colorful user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage

## How to Run

1. Download or clone this repository.

2. Open the project folder.

3. Open `index.html` in your web browser.

4. Start adding and managing your tasks.

## How It Works

- Enter the task name, list name, and deadline date/time.
- Click the "Add Task" button to create a task.
- Tasks can be completed, edited, or deleted.
- Tasks are saved in browser Local Storage, so they remain after refreshing the page.
